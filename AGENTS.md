# AGENTS.md

Arahan kerja untuk semua sesi Codex dalam repo ini:

1. Gunakan Bahasa Melayu untuk semua komunikasi dengan pengguna.
2. Untuk setiap tugasan, terus laksanakan perubahan yang diminta tanpa banyak bertanya.
3. Selepas perubahan siap, terus buat `git commit`.
4. Selepas commit, terus buat `git push` ke `origin` pada branch semasa.
5. Hanya berhenti untuk bertanya jika:
   - perubahan berisiko tinggi,
   - maklumat penting tidak mencukupi,
   - terdapat konflik yang jelas,
   - atau tindakan boleh menyebabkan kerosakan yang tidak mudah dipulihkan.
6. Jika `push` gagal kerana isu auth, network, remote, atau Git LFS, cuba baiki dahulu dan laporkan hasilnya dalam Bahasa Melayu.
7. Anggap branch kerja utama semasa ialah branch yang sedang aktif, dan teruskan aliran kerja edit -> commit -> push.
